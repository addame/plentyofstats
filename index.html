About the Project

Plenty of Stats is a project to scrape the popular internet dating site Plenty of Fish using Ruby and why's Hpricot library.

Using the 'number of results' Plenty of Fish provides on each search request, it's possible to query each age and sex and build a list of results by age and city.

I started with a base search, no user attributes, to get the number of Plenty of Fish users for each gender and age 18 to 65, in each of the top 15 cities(according to Wikipedia). After building the base results, I scraped for drinking, smoking, and drug use(combining 'social' and 'heavy' use).

The Data, and Weird Anomalies

  • Download the data and the Ruby code used to scrape it on github

Feel free to run the data through your own report and see what you get. You can pick up the data, in CSV and SQLite format in the github repo above.

Obviously this data is purely for entertainment, as I had to massage some of the city data to get it under the 700 result limit imposed by Plenty of Fish. In the case of LA and NY, I only queried users which had chosen to list their height. And in London, I picked users with average body types. I can't figure out what's up with the drinking in Chitown, LA, NYC, and DC. I've checked the site by hand and confirmed the numbers.

And of course at the 55+ range the data is bollocks, since the number of users in that age range are few and far between.
