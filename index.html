About the Project

Plenty of Stats is a project to scrape the popular internet dating site Plenty of Fish. Using the 'number of results' Plenty of Fish provides on each search request, it's possible to build a list of ages with their matching results.

I started with a base search, no attributes, just the number of Plenty of Fish users for each gender, each age 18 to 65, in each of the top 15 cities(according to Wikipedia). After building the base results, I scraped for drinking, smoking, and drug use.
